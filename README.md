## Kafka - Kafka Mini Project

## Summary
SpringBoard Project. Use Dockerized Kafka Zookeeper and Broker, Producer and Generator.

## Description
Simulate banking transactions, and parse into two topics "legit" and "fraud" based on conditions. 

## Technologies
- Python 3.6
- Ubuntu
- Docker
- Kafka
